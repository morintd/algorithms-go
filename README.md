# algorithms-go

An attempt at basic but elegant implementation of multiple algorithms in Go.

## Includes

- Binary Search List
- Binary Search Tree (BST), using recursivity
  - `Search`
  - `Add`/`Delete`
  - `FindMin`/`FindMax`
- Bubble Sort
- Queue
- Stack
- Two Crystall Balls problem

## Commands

```sh
$ make test
```
